To run this project you need to have:

* Ruby 2.3.3
* Rails 5.0.2
* PostgreSQL

## Clone this project
  ```
  $ git clone https://github.com/wilkinsondarolt/contacts.git
  ```

## Enter project folder
  ```
  $ cd catarse
  ```

## Install the gems
  ```
  $ bundle install
  ```

## Create and seed the database
  ```
  $ rake db:create db:migrate
  ```

## Run Rails server
  ```
  $ rails server
  ```
